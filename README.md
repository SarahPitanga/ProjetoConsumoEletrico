# #Calculadora de Consumo Energético  – Calcula o consumo médio mensal de aparelhos domésticos
Este projeto em Python foi criado para praticar o ciclo **Entrada → Processamento →
Saída**.
O programa solicita o **nome do aparelho**, **potência**, **horas_dias**, **multiplica por 30 e divide por 1000** calcula a média e mostra o valor médio que esse aparelho gasta ao longo do mês
