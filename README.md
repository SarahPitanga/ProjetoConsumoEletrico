#  🔌 #Calculadora de Consumo 💲 Energético  – Calcula o consumo médio mensal de aparelhos domésticos 🔌
⚡ Calculadora de Energia 🔌
O que é este projeto?
🐍 Um programa simples em Python para calcular quanto um aparelho gasta de energia por mês. 🐍

Como funciona?
O programa pede três informações:

Nome do aparelho.

⚡Potência (em Watts). ⚡

🕐Horas de uso por dia. 🕐

Ele usa a fórmula abaixo para calcular o consumo:
Consumo = Potencia * Horas * 30 /1000

💻 Exemplo de Saída
Ao executar o programa, você verá algo como:

Aparelho: Geladeira 🧊

Consumo estimado: 45.0 kWh/mês ⚡

📊 Resultado:

Custo estimado: R$ 33.75/mês 💲
